# 🧠 AI News Generator

This project automatically:
- Fetches latest news
- Uses AI to summarize it
- Sends updates via email

## 🚀 Tech Stack
- Python
- NewsAPI
- Hugging Face Transformers
- SMTP (Email Automation)

## ⚙️ How It Works
1. Fetch news from API
2. Combine headlines
3. Generate AI summary
4. Send email update
